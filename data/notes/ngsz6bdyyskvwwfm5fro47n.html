🧳 旅行 Vlog 拍摄 & 剪辑 Checklist（拉片总结版）

目标：为剪辑服务，为叙事服务，为观众体验服务
原则：可以设计，但保持真实感受

一、出发前 / 出发瞬间（建立「要旅行了」的感觉）

  • 🎫 机票正面（多张、瞬间）
  • 🎒 行李箱（收拾 / 拉链 / 推着走）
  • 🚪 开门 / 关门（可做黑 → 白转场）
  • ✋ 手指在地图上划路线
  • ⏱️ 快速拍一些 1 秒就能用的画面（给片头用）

👉 用象征物代替解释，让观众秒懂「出发」

二、机场（空间感 + 设计感）

空镜 & 氛围

  • 🏢 机场空镜（走廊 / 候机区 / 指示牌）
  • 🪟 朝窗外看的镜头（情绪暂停）
  • ✈️ 飞机停靠 / 起飞相关画面

设计好的互动片段（可提前想）

  • 🤏 重复性小动作（比如固定姿势、固定流程）
  • 🙂 可爱的、观众能预期的小桥段
  • 🧠 可提前准备的聊天内容 / 知识点

👉 vlog = 有设计的真实，而不是纯随机记录

三、飞机 / 路途（“路有多远”）

  • 🗺️ 航线 / 地图 / 距离展示
  • 🪟 飞机窗外
  • 🛫 起飞 / 降落的情绪节点
  • 🧳 行李再次出现（状态变化）

👉 把抽象距离，变成看得见的东西

四、到达 & 旅途中（日常 + 情绪）

情绪 & 感受

  • 🗣️ 口播：把镜头当成观众
  • 💭 讲“感受”和“当下”，不只是发生了什么
  • 👀 看窗外 / 走路中的思考镜头

内容价值

  • 💡 旅行中的生活小窍门
  • 📌 实用经验 / 小技巧
  • 🩺 像“旅游大夫”一样的知识视角（可作为一个固定身份）

五、拍摄技巧（为剪辑留余地）

同一主体，多覆盖

  • 🎥 同一个东西：远景 / 中景 / 近景
  • 👤 人物近景 + 环境远景

可爱 & 对照

  • 🧍 人物姿势
  • 🐕 动物 / 他人相同姿势（制造相似感）

六、B-roll 素材（剪辑自由度）

  • ✋ 手部特写
  • 👀 眼神 / 表情
  • 📦 物件特写（食物、票据、工具）
  • 🚶 行走 / 推门 / 拿起 / 放下

👉 你在说话时，画面不一定是你

七、转场 & 标题出口

  • 🚪 开门 / 进门
  • 🌗 黑 → 白 / 明 → 暗
  • 🔄 场景 → 地图 → 下个场景
  • 🎬 可单独当“句号/冒号”的自然动作

八、剪辑逻辑提醒（拍的时候心里要有）

  • 🎞️ 类似镜头剪在一起（个人 → 群体 → 远景）
  • ⚡ 片头快剪（1 秒左右一镜）
  • 🧭 帮观众保持方向感，不迷路
  • 🎵 留情绪空间给音乐和旁白

核心自检问题：

  • 如果我是观众，我会不会想继续看？
  • 这个画面，是信息、情绪，还是转场？
  • 我有没有为剪辑提前准备选择权？
